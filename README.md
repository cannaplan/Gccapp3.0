# Gaia Commons Council Dashboard v3.0 (70/30 Hybrid)

MN statewide school food security: 1,200 greenhouses (12M sqft), 70% hydro towers (150k units), 30% soil beds. Feeds 900k students via $5B endowment.

## Quick Start
```bash
npm install
npm run dev
```

Live: https://gccapp3-0.replit.app | GitHub: https://github.com/JfreeDev/gccapp3.0


## Key Features
- 70/30 Greenhouse Metrics (/greenhouse/7030)
- BOM Calculator w/ Agrotonomy Quotes (/bom)
- Investor Dashboard (LCOF/BCR, Sensitivity)
- Social Assets (/social, /flyers)
- DAO Petition Tracker (/dao)

## Tech
- React + react-router-dom v6+
- Shadcn UI, Error Boundaries
- Print-optimized Exports

## Deploy
```bash
npm install
npm run build
npx serve -s build
```
